# Locale JSON (reference)

Canonical translation bundles live in **`packages/web/src/locales/*.json`** and are imported by **`packages/web/src/i18n.js`**.

This folder exists so contributor docs that mention `src/i18n/locales/` resolve to a real path. **Edit `src/locales/*.json`** for runtime i18n changes.
